# PopupBlocker

一个轻量级的弹窗拦截软件

## 使用教程

emmm，其实很简单这里只说明特殊的地方

* 正常情况下不要打开日志，因为日志只暂存在内存里
* Class/Title指的是窗口拦截方式，一般情况下国外软件用Class，国内软件用Title
* Class是默认的，你得手动调整为Title
* 进程拦截顾名思义，不过应该很少用到
